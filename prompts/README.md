# Scoring prompts (frozen, paper-v1)

The exact Jinja2 prompt templates used by the Mizaan primary scorer, independent auditor, and amendment drafter when the released results were generated (git tag `paper-v1`). Rubric dimensions are injected at render time. Reproducing the paper uses these prompts unchanged; later product work (e.g. auditor calibration, ISSUE-001) diverges on commits AFTER `paper-v1`.
